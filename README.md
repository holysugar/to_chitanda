# ToChitanda

Integer#to_chitanda

## Installation

Add this line to your application's Gemfile:

    gem 'to_chitanda'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_chitanda

## Usage

```ruby
1000.to_chitanda #=> "千反田える"
```
